## 2025-05-15 - [Layout Broken by Copied CSS]
**Learning:** Copying CSS rules from specialized components (like rotated chart labels) to generic utility classes (like .empty-state) can cause severe layout regressions if not carefully pruned.
**Action:** Always verify empty states and edge cases visually when modifying shared layout components.

## 2025-05-15 - [Consistency in Interactive Elements]
**Learning:** Users expect similar data types (e.g., ticker symbols) to have consistent interaction patterns across different tabs.
**Action:** Proactively align interaction patterns (like click-to-copy) when discovering fragmented implementations of the same concept.
